# 2026 인공지능 부트캠프 프로젝트
## 주제 : 중고차 가격 예측
### 국내 보유 차량이 증가함에 따라 중고차량 시장또한 확장 가격책정이 중요함 따라서 사용자가 예측가격을 보고 다른 차량들과 비교 할수 있는 기준이 필요 
### 차량연식, 주행거리등 다양한 요소에 따라 중고차 가격 책정에 있어 소비자가 책정하기 어려움 따라서 인공지능을 활용하여 가격 예측 및 추천 기능을 제공 

## [사용기술&데이터]
### 데이터 수집: kaggle (Used_Car_Price_Data.csv)
### 언어: python
### 모델: RandomForestRegressor (Scikit-learn), AI: OpenAI API (GPT)
### 백엔드: FastAPI, Uvicorn
### 프론트 앤드: Streamlit, HTML/CSS (Custom CSS)
### 데이터 처리: Pandas, NumPy
### 버전관리: Git&GitHub
### 배포 :  Docker, Render

## [기능설명]
### 차량 정보 입력 및 가격 예측
### 사용자가 차량의 연식, 주행거리, 연료 종류, 신차 가격 등의 정보를 입력하면 RandomForest 회귀모델을 이용하여 예상 중고차 가격을 실시간으로 예측

### AI 기반 결과 분석
### OpenAI API를 활용하여 예측된 가격의 적정성과 차량 상태를 종합적으로 분석하고, 사용자가 이해하기 쉬운 형태로 설명을 제공

### 유사 차량 추천 기능
### 예측된 가격과 비슷한 가격대의 차량을 데이터셋에서 검색하여 차량명, 가격, 이미지 등의 정보를 함께 제공

### FastAPI와 Streamlit을 연동하여 백엔드와 프론트엔드를 분리

### Streamlit의 Custom CSS를 적용하여 UI를 구현

## [문제점 및 개선 방안]
### 해외 중고차 데이터셋을 사용하여 국내 차량 환경을 충분히 반영하지 못하는 한계가 있으며 해외 중고차 데이터셋도 데이터량이 많지 않았음. (차량 요소 컬럼이 통일되어 있지 않음) 
### 향후 국내 중고차 데이터를 추가하여 예측 정확도를 향상과 사고 이력 등의 정보를 반영 추천 알고리즘을 개선하여 사용자 맞춤형 차량 추천 기능을 강화

## [활용방안 및 기대효과]
## 사용자가 차량의 적정가격을 손쉽게 확인하는데에 활용가능하며 AI 기반 가격 분석과 추천 기능을 통해 합리적인 구매가능 또한 웹 기반 서비스로 구현하여 별도의 프로그램 설치 없이 누구나 쉽게 이용가능
